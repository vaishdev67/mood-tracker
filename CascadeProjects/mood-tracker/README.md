# Mood Tracker Web App

A simple and beautiful mood tracking application that helps you and your friends track daily moods and emotions.

## Features

- Track your daily mood (Happy, Sad, Angry, Meh)
- Add notes about your day
- View weekly mood statistics
- See common words from your notes
- Get personalized tips based on your mood patterns
- Responsive design that works on all devices

## How to Deploy to Netlify

1. **Create a GitHub account** (if you don't have one) at [github.com](https://github.com/)
2. **Create a new repository** on GitHub
3. **Upload the files** to your GitHub repository
4. **Sign up for Netlify** at [app.netlify.com](https://app.netlify.com/)
5. **Click "New site from Git"** in Netlify
6. **Choose GitHub** as your Git provider
7. **Select your repository** that contains the mood tracker code
8. **Configure the build settings** (Netlify will auto-detect them)
9. **Click "Deploy site"**
10. **Share the URL** with your friends!

## How to Use

1. Visit your deployed website
2. Select how you're feeling today
3. Add a note about your day
4. View your mood history and statistics
5. Share the link with friends to track together!

## Data Privacy

All mood entries are stored locally in your browser's local storage. No personal data is collected or stored on any server.

## Customization

You can customize the app by editing the following files:
- `styles.css` - Change colors, fonts, and layout
- `index.html` - Modify the structure and content
- `app.js` - Add new features or modify existing functionality
